[English]
Hello, I am Ozcan Bayram. I am 20 years old and I am a computer programming student at Bandırma Onyedi Eylül University. I'm studying to be a web designer and mobile developer.
While learning software since my high school years, I improved myself in the basics of programming with the C# language, learned basic Web development using Html, Css, Php, and learned the basics of Android by developing mobile applications with Java. Now I am developing projects by learning the necessary languages for mobile and web development.
I am currently working mainly on the Kotlin language. I develop projects individually and as a team. At the end of this university semester, my team and I are working with excitement to create an impressive project. I can't wait to add this website project we worked on using Html, Css, Js, Php and Sql to my portfolio.
I continue my work passionately to share my social media project called 'Gezle', which I am developing individually using Kotlin for Android stores. I love my job and try to learn new things every day.
You can review my projects on my GitHub profile, support me and reach me through the links below for more information about me.
